# 🎬 CineMatch — Movie Recommendation System

A content-based, collaborative filtering, and hybrid movie recommendation engine built with Python, Jupyter Notebook, and Flask.

---

## ✨ Features

- 🎭 **Content-Based Filtering** — Recommends movies similar to one you liked, based on genre, director, cast, and description using TF-IDF and cosine similarity
- 👥 **Collaborative Filtering** — Recommends movies based on what users with similar taste enjoyed
- 🔀 **Hybrid Recommender** — Blends both methods for the most personalised results
- 🌐 **Web Interface** — Clean Flask-powered UI with autocomplete search and interactive controls
- 📊 **Data Visualisation** — EDA charts and similarity heatmaps in the Jupyter notebook

---

## 🚀 Getting Started

1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `python app.py`
3. Open browser at: `http://127.0.0.1:5000`

---

## 🧠 How It Works

- **Content-Based** — TF-IDF + cosine similarity on movie features
- **Collaborative** — User-user similarity matrix from ratings
- **Hybrid** — 50/50 weighted blend of both methods
